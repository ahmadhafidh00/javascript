## Calculate Total Money

### Deskripsi

Seseorang memiliki banyak tempat penyimpanan uang yang direpresentasikan dalam bentuk objek. Tugasnya adalah membuat sebuah fungsi untuk menghitung total uang dari orang tersebut.

### Input

```javascript
let person = {
  kantong: 10000,
  bank: 200000,
  brankas: 350000,
};
```

### Output

Total uang orang tersebut adalah 560000
